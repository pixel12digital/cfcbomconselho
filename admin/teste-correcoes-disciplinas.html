🧪 Teste das Correções das Disciplinas

📋 Instruções de Teste

Problema 1: Total de horas não diminui para todas as disciplinas

Problema 2: Disciplinas selecionadas não aparecem na próxima tela

✅ Correções Implementadas

  • Event Listener do Formulário: Adicionado interceptação do submit para chamar `criarTurmaComDisciplinas()`
  • Cálculo de Horas: Melhorado para considerar TODAS as disciplinas selecionadas
  • Salvamento de Disciplinas: Corrigido fluxo para salvar disciplinas antes de redirecionar
  • Redirecionamento: Corrigido para usar o ID correto da turma criada

🧪 Testes a Executar

Teste 1: Cálculo do Total de Horas

  1. Acesse: Nova Turma - Etapa 1
  2. Selecione um curso (ex: Formação 45h)
  3. Adicione a primeira disciplina
  4. Verificar: Total de horas deve diminuir
  5. Adicione a segunda disciplina
  6. Verificar: Total de horas deve diminuir novamente
  7. Adicione mais disciplinas
  8. Verificar: Cada disciplina deve diminuir o total

Teste 2: Disciplinas na Próxima Tela

  1. Na mesma página, adicione 3-4 disciplinas
  2. Preencha os dados obrigatórios (nome, datas, etc.)
  3. Clique em "Próxima Etapa: Agendamento →"
  4. Verificar: Deve redirecionar para etapa 2
  5. Verificar: Select "Disciplina" deve mostrar as disciplinas adicionadas
  6. Verificar: Console deve estar limpo (F12)

🔧 Scripts de Migração

Se as disciplinas não aparecerem na etapa 2, execute:

  • 🔧 Verificar Migração
  • 🧪 Teste de Conexão

📊 Logs para Verificar

Abra o console (F12) e verifique se aparecem:

  • ✅ 🎯 Formulário de turma submetido - chamando criarTurmaComDisciplinas
  • ✅ 🔍 [PÁGINA PRINCIPAL] Encontradas X disciplinas
  • ✅ 📊 [PÁGINA PRINCIPAL] Disciplina X: Nome (Xh)
  • ✅ ✅ Turma criada: [ID]
  • ✅ 💾 Salvando disciplinas selecionadas: [array]
  • ✅ 🎯 Redirecionando para etapa 2 com turma_id: [ID]
