Teste da Topbar Unificada

Esta página testa a implementação da topbar unificada do Sistema CFC. Verifique se todos os elementos estão funcionando corretamente.

Layout

Topbar fixa no topo com altura de 64px, sem gaps ou espaços em branco.

Distribuição

Logo à esquerda, busca central fluida, notificações e perfil à direita.

Comportamento Sticky

Topbar permanece fixa no topo durante a rolagem da página.

Busca Global

Campo de busca centralizado com debounce de 300ms e sugestões em dropdown.

Notificações

Sino com badge de contagem e dropdown com últimas notificações.

Perfil

Menu do usuário com dropdown contendo opções de perfil e logout.

Responsividade

Layout adaptável para diferentes tamanhos de tela (desktop, tablet, mobile).

Acessibilidade

Navegação por teclado, labels ARIA e foco visível em todos os elementos.

Instruções de Teste

1. Layout e Espaçamento

  • Verifique se não há espaço em branco entre a topbar e o conteúdo
  • Confirme que a topbar tem altura fixa de 64px
  • Teste se a sidebar está posicionada corretamente (70px de largura)

2. Funcionalidades

  • Busca: Digite pelo menos 3 caracteres e aguarde 300ms
  • Notificações: Clique no sino para ver o dropdown
  • Perfil: Clique no avatar para ver o menu
  • Atalhos: Pressione "/" para focar na busca

3. Responsividade

  • Redimensione a janela para testar breakpoints
  • Verifique se os elementos se adaptam corretamente
  • Teste em dispositivos móveis (largura ≤ 768px)

4. Acessibilidade

  • Navegue usando apenas o teclado (Tab, Enter, Escape)
  • Verifique se há indicadores de foco visíveis
  • Teste leitores de tela se disponível
