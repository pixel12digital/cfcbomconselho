🧪 Teste do Menu Mobile - V2

Este é um arquivo de teste para validar o funcionamento do menu mobile reorganizado. Redimensione a janela para ≤768px para ver:

  • ✅ Topbar compacta (logo + hambúrguer apenas)
  • ✅ Busca movida para abaixo do header
  • ✅ Filtros colapsáveis
  • ✅ Hierarquia e respiro adequados

Teste de Funcionalidades - Menu

Clique no botão hambúrguer (aparece em telas ≤768px) para testar:

  • ✅ Abertura/fechamento do drawer
  • ✅ Animação do ícone hambúrguer → X
  • ✅ Navegação por teclado (TAB, ESC)
  • ✅ Foco preso no drawer
  • ✅ Clique fora para fechar
  • ✅ Scroll lock (body não rola com menu aberto)
  • ✅ Z-index correto (WhatsApp flutuante oculto)

Teste de Funcionalidades - Busca e Filtros

Teste a nova busca mobile (aparece em telas ≤768px):

  • ✅ Campo de busca com placeholder curto
  • ✅ Botão de busca/lixeira dinâmico
  • ✅ Filtros colapsáveis (botão "Filtros")
  • ✅ Aplicar/limpar filtros
  • ✅ Chips de filtros ativos
  • ✅ Remoção individual de filtros
  • ✅ Animações suaves (200-250ms)
