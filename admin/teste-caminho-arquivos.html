🔧 Teste - Caminho dos Arquivos

✅ PROBLEMA CORRIGIDO:
O erro de caminho dos arquivos foi corrigido no arquivo admin/pages/turmas-teoricas.php
🐛 Problema Identificado

O erro ocorria na linha 24 do arquivo turmas-teoricas.php:

Erro Original:
Warning: require_once(/cfc-bom-conselho/admin/pages/includes/database.php): Failed to open stream: No such file or directory

Causa: O caminho estava incorreto - estava tentando acessar /admin/pages/includes/ quando deveria ser /includes/

🔧 Correção Aplicada

Alterado o caminho dos includes de:

require_once $base_path . '/includes/config.php';
require_once $base_path . '/includes/database.php';
require_once $base_path . '/includes/auth.php';

Para:

require_once '../../includes/config.php';
require_once '../../includes/database.php';
require_once '../../includes/auth.php';
📁 Estrutura de Arquivos

A estrutura correta é:

cfc-bom-conselho/
├── includes/
│ ├── config.php
│ ├── database.php
│ └── auth.php
└── admin/
└── pages/
└── turmas-teoricas.php

Para ir de admin/pages/ até includes/, precisamos subir dois níveis: ../../includes/

🧪 Como Testar
  1. Acesse a página: admin/pages/turmas-teoricas.php
  2. Verifique se não há erros: A página deve carregar normalmente
  3. Teste o modal de disciplinas: Clique em "Gerenciar Disciplinas"
  4. Teste os botões de fechar: Tanto o "X" quanto o "Fechar" devem funcionar
⚠️ Se ainda houver problemas:
  • Verifique se o XAMPP está rodando
  • Verifique se os arquivos config.php, database.php e auth.php existem na pasta includes/
  • Verifique as permissões dos arquivos
  • Verifique se há erros no log do PHP
