1. Validação Array.isArray()

Problema: Código assumia que `data.disciplinas` sempre seria um array.

Solução: Validação explícita com `Array.isArray()`

// ANTES (sem validação) if (data.success && data.disciplinas) { data.disciplinas.forEach(disciplina => { // ... }); } // DEPOIS (com validação) if (data.success && data.disciplinas && Array.isArray(data.disciplinas)) { data.disciplinas.forEach(disciplina => { // ... }); } else { console.error('❌ [API] Dados inválidos:', data); console.error('❌ [API] É array?', Array.isArray(data.disciplinas)); }
