📋 Problema Identificado

Turma 6 criada: ✅ "Formação de Condutores - Teste"

Disciplinas salvas: ❌ Nenhuma disciplina encontrada no banco

Causa: Event listener do formulário pode não estar funcionando

🧪 Teste do Event Listener

Para testar se o event listener está funcionando:

  1. Acesse: Nova Turma
  2. Abra o console (F12)
  3. Adicione algumas disciplinas
  4. Preencha os dados obrigatórios
  5. Clique em "Próxima Etapa: Agendamento →"
  6. Verifique no console: Deve aparecer "🎯 Formulário de turma submetido"
