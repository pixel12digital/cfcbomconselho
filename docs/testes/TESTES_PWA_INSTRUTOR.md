# TESTES – PWA Instrutor

## Objetivo

Este checklist valida as funcionalidades do **PWA do Instrutor**, incluindo dashboard, visualização de aulas do dia, início/finalização de aulas práticas, registro de km, marcação de presença teórica (se implementado) e demais funcionalidades disponíveis para o instrutor.

## Cenários Principais (rascunho)

- [ ] Cenário 1 – Login e acesso ao dashboard do instrutor
- [ ] Cenário 2 – Visualização de agenda de aulas do dia
- [ ] Cenário 3 – Início de aula prática (registro de km inicial, horário)
- [ ] Cenário 4 – Finalização de aula prática (registro de km final, carga horária)
- [ ] Cenário 5 – Marcação de falta do aluno (quando aplicável)
- [ ] Cenário 6 – Visualização de dados do aluno (dados essenciais durante aula)
- [ ] Cenário 7 – Registro de presença teórica via PWA (se implementado)
- [ ] Cenário 8 – Responsividade mobile (layout em dispositivos móveis)
