# TESTES – Módulo de Alunos (Admin)

## Objetivo

Este checklist valida as funcionalidades do **módulo de Alunos** no painel administrativo, incluindo cadastro, edição, visualização de histórico, matrículas e todas as funcionalidades relacionadas à gestão de alunos.

## Cenários Principais (rascunho)

- [ ] Cenário 1 – Cadastro de novo aluno (dados pessoais, documentos)
- [ ] Cenário 2 – Edição de aluno existente
- [ ] Cenário 3 – Visualização de histórico completo do aluno
- [ ] Cenário 4 – Criação/edição de matrícula
- [ ] Cenário 5 – Visualização de progresso teórico do aluno
- [ ] Cenário 6 – Visualização de progresso prático do aluno
- [ ] Cenário 7 – Visualização de resumo financeiro do aluno
- [ ] Cenário 8 – Visualização de exames/provas do aluno
- [ ] Cenário 9 – Filtros e busca de alunos
- [ ] Cenário 10 – Inativação de aluno

## Modelo de Caso de Teste

Para cada cenário, quando formos detalhar:

- **Cenário:** `<nome>`
- **Pré-condições:** `<o que precisa existir - ex.: CFC cadastrado, etc.>`
- **Passos:**
  1. `<passo 1>`
  2. `<passo 2>`
  3. `<passo 3>`
- **Resultado esperado:** `<comportamento esperado após os passos>`
- **Status:** [ ] Pendente / [x] Aprovado / [ ] Falhou
